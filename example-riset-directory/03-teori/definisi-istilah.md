# Definisi Istilah

## Pendahuluan

Dokumen ini berisi definisi istilah-istilah yang digunakan dalam penelitian **"Sistem Absensi Guru Berbasis QR Code Menggunakan Framework Laravel"**. Tujuan penyusunan glosarium ini adalah memberikan pemahaman yang seragam terhadap istilah teknis yang digunakan selama proses penelitian, pengembangan sistem, maupun penyusunan laporan.

---

# Daftar Istilah

| No. | Istilah | Definisi |
|-----:|---------|----------|
| 1 | Sistem Informasi | Sistem yang mengolah data menjadi informasi untuk mendukung kegiatan operasional dan pengambilan keputusan. |
| 2 | Absensi | Proses pencatatan kehadiran seseorang dalam suatu kegiatan pada waktu tertentu. |
| 3 | Guru | Tenaga pendidik yang menjadi objek utama dalam sistem absensi yang dikembangkan. |
| 4 | QR Code | Barcode dua dimensi yang digunakan sebagai media identifikasi guru saat melakukan absensi. |
| 5 | Laravel | Framework PHP berbasis MVC yang digunakan dalam pengembangan aplikasi. |
| 6 | PHP | Bahasa pemrograman server-side yang digunakan untuk membangun logika aplikasi. |
| 7 | MySQL | Sistem Manajemen Basis Data (DBMS) yang digunakan untuk menyimpan data aplikasi. |
| 8 | Bootstrap | Framework CSS yang digunakan untuk membangun tampilan antarmuka aplikasi agar responsif dan menarik. |
| 9 | Database | Kumpulan data yang tersusun secara sistematis sehingga mudah diakses dan dikelola. |
| 10 | Framework | Kerangka kerja yang menyediakan struktur dan komponen untuk mempermudah pengembangan aplikasi. |
| 11 | MVC (Model-View-Controller) | Arsitektur pengembangan perangkat lunak yang memisahkan logika aplikasi, tampilan, dan pengelolaan data. |
| 12 | Model | Bagian dari arsitektur MVC yang bertugas mengelola data dan berinteraksi dengan database. |
| 13 | View | Bagian dari MVC yang menampilkan antarmuka kepada pengguna. |
| 14 | Controller | Bagian dari MVC yang menghubungkan Model dengan View serta mengatur alur proses aplikasi. |
| 15 | CRUD | Singkatan dari Create, Read, Update, Delete, yaitu operasi dasar dalam pengelolaan data. |
| 16 | Authentication | Proses verifikasi identitas pengguna sebelum memperoleh akses ke sistem. |
| 17 | Authorization | Proses pemberian hak akses kepada pengguna sesuai dengan perannya. |
| 18 | Migration | Fitur Laravel yang digunakan untuk membuat dan mengelola struktur tabel database. |
| 19 | Seeder | Fitur Laravel yang digunakan untuk mengisi data awal ke dalam database. |
| 20 | Eloquent ORM | Object Relational Mapping bawaan Laravel yang mempermudah pengelolaan database menggunakan objek. |
| 21 | Route | Mekanisme pada Laravel yang menghubungkan URL dengan fungsi atau controller tertentu. |
| 22 | Middleware | Komponen Laravel yang mengatur proses sebelum permintaan diproses oleh aplikasi, seperti autentikasi pengguna. |
| 23 | Blade Template | Mesin template bawaan Laravel yang digunakan untuk membangun tampilan aplikasi secara dinamis. |
| 24 | Dashboard | Halaman utama aplikasi yang menampilkan ringkasan informasi dan statistik sistem. |
| 25 | Web Server | Perangkat lunak yang menjalankan aplikasi web dan melayani permintaan dari pengguna. |
| 26 | XAMPP | Paket perangkat lunak yang berisi Apache, PHP, dan MySQL untuk menjalankan aplikasi secara lokal. |
| 27 | Apache | Web server yang digunakan untuk menjalankan aplikasi Laravel pada lingkungan pengembangan. |
| 28 | Visual Studio Code | Editor kode sumber yang digunakan dalam proses pengembangan aplikasi. |
| 29 | Black Box Testing | Metode pengujian perangkat lunak yang berfokus pada fungsi sistem tanpa melihat kode program. |
| 30 | Waterfall | Metode pengembangan perangkat lunak yang dilakukan secara bertahap dan berurutan. |

# Istilah Khusus pada Penelitian

Beberapa istilah berikut merupakan istilah yang secara langsung berkaitan dengan implementasi Sistem Absensi Guru Berbasis QR Code.

| Istilah | Penjelasan |
|----------|------------|
| Generate QR Code | Proses pembuatan QR Code unik untuk setiap data guru. |
| Scan QR Code | Proses membaca QR Code menggunakan kamera perangkat sebagai media pencatatan absensi. |
| Absensi Masuk | Proses pencatatan waktu kehadiran guru saat mulai bekerja. |
| Absensi Pulang | Proses pencatatan waktu kepulangan guru setelah menyelesaikan kegiatan mengajar. |
| Data Guru | Informasi yang memuat identitas guru seperti NIP, nama, jabatan, nomor telepon, dan alamat email. |
| Laporan Absensi | Rekapitulasi data kehadiran guru yang dapat ditampilkan maupun dicetak oleh administrator. |
