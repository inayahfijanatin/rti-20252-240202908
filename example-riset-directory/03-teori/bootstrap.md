# Bootstrap

## Pendahuluan

Antarmuka pengguna (User Interface/UI) merupakan salah satu aspek penting dalam pengembangan aplikasi berbasis web. Tampilan yang menarik, responsif, dan mudah digunakan akan meningkatkan kenyamanan pengguna dalam berinteraksi dengan sistem. Untuk mempermudah proses pengembangan antarmuka, banyak pengembang memanfaatkan framework CSS, salah satunya adalah Bootstrap.

Bootstrap merupakan framework front-end yang menyediakan kumpulan komponen antarmuka siap pakai sehingga proses pembuatan halaman web menjadi lebih cepat, konsisten, dan responsif. Dengan menggunakan Bootstrap, pengembang tidak perlu membuat seluruh tampilan dari awal karena telah tersedia berbagai komponen seperti tombol, tabel, formulir, kartu (*card*), navigasi, hingga sistem grid.

Pada penelitian ini, Bootstrap digunakan untuk membangun tampilan Sistem Absensi Guru Berbasis QR Code agar memiliki antarmuka yang modern, responsif, dan mudah digunakan.

---

# Pengertian Bootstrap

Bootstrap adalah framework CSS open source yang digunakan untuk membangun antarmuka aplikasi web secara cepat dan responsif. Framework ini dikembangkan oleh Mark Otto dan Jacob Thornton di Twitter pada tahun 2011.

Bootstrap menyediakan berbagai komponen antarmuka yang telah dilengkapi dengan CSS dan JavaScript sehingga dapat langsung digunakan dalam proses pengembangan aplikasi web.

Menurut dokumentasi resmi Bootstrap, framework ini dirancang untuk mempermudah pengembangan website yang responsif (*mobile-first*) dengan menyediakan sistem grid, komponen antarmuka, serta utilitas CSS yang lengkap.

---

# Sejarah Bootstrap

Bootstrap pertama kali dikembangkan oleh **Mark Otto** dan **Jacob Thornton** sebagai framework internal di Twitter.

Pada tahun 2011 Bootstrap dirilis sebagai proyek open source dan sejak saat itu menjadi salah satu framework front-end paling populer di dunia.

Saat ini Bootstrap telah mengalami berbagai pembaruan hingga versi Bootstrap 5 yang digunakan pada penelitian ini. Bootstrap 5 tidak lagi bergantung pada jQuery sehingga memiliki performa yang lebih ringan dan modern.

---

# Karakteristik Bootstrap

Bootstrap memiliki beberapa karakteristik sebagai berikut:

- Bersifat open source.
- Menggunakan konsep **Mobile First**.
- Responsif pada berbagai ukuran layar.
- Memiliki sistem grid 12 kolom.
- Menyediakan banyak komponen siap pakai.
- Mudah dikustomisasi.
- Didukung dokumentasi yang lengkap.

---

# Komponen Bootstrap

Bootstrap menyediakan berbagai komponen antarmuka yang dapat digunakan dalam pengembangan aplikasi.

### 1. Grid System

Grid System digunakan untuk mengatur tata letak halaman agar responsif pada berbagai ukuran layar.

### 2. Navbar

Navbar digunakan sebagai menu navigasi utama aplikasi.

### 3. Card

Card digunakan untuk menampilkan informasi dalam bentuk panel yang rapi, seperti statistik pada dashboard.

### 4. Table

Komponen tabel digunakan untuk menampilkan data secara terstruktur, misalnya daftar guru dan data absensi.

### 5. Form

Bootstrap menyediakan berbagai elemen formulir seperti input, select, checkbox, radio button, dan textarea dengan tampilan yang konsisten.

### 6. Button

Komponen tombol digunakan untuk menjalankan berbagai aksi dalam aplikasi, seperti menambah data, mengubah data, menghapus data, maupun mencetak laporan.

### 7. Modal

Modal digunakan untuk menampilkan dialog atau konfirmasi tanpa berpindah halaman.

### 8. Alert

Alert digunakan untuk menampilkan notifikasi kepada pengguna, seperti pesan berhasil, gagal, atau peringatan.

# Kelebihan Bootstrap

Bootstrap memiliki beberapa kelebihan, antara lain:

- Mempercepat proses pengembangan antarmuka.
- Tampilan responsif pada berbagai perangkat.
- Dokumentasi lengkap dan mudah dipelajari.
- Memiliki banyak komponen siap pakai.
- Konsisten pada berbagai browser.
- Mudah dipadukan dengan Framework Laravel.
- Mendukung pengembangan aplikasi modern.

---

# Kekurangan Bootstrap

Selain memiliki berbagai kelebihan, Bootstrap juga memiliki beberapa kekurangan, yaitu:

- Tampilan dapat terlihat mirip dengan aplikasi lain jika tidak dikustomisasi.
- Mengandung banyak komponen yang mungkin tidak seluruhnya digunakan.
- Membutuhkan penyesuaian CSS apabila ingin menghasilkan desain yang benar-benar unik.

# Penerapan Bootstrap dalam Penelitian

Pada penelitian ini, Bootstrap digunakan untuk membangun antarmuka Sistem Absensi Guru Berbasis QR Code. Beberapa halaman yang memanfaatkan Bootstrap antara lain:

- Halaman Login
- Dashboard
- Data Guru
- Generate QR Code
- Scan QR Code
- Data Absensi
- Laporan Absensi
- Form Tambah Guru
- Form Edit Guru

Selain itu, Bootstrap dimanfaatkan untuk membuat tampilan aplikasi lebih responsif sehingga dapat diakses dengan baik melalui komputer maupun perangkat bergerak.
