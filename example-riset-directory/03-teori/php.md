# PHP (Hypertext Preprocessor)

## Pendahuluan

Perkembangan aplikasi berbasis web tidak terlepas dari penggunaan bahasa pemrograman yang mampu mengelola logika aplikasi, memproses data, serta berinteraksi dengan basis data. Salah satu bahasa pemrograman yang paling banyak digunakan dalam pengembangan aplikasi web adalah **PHP (Hypertext Preprocessor)**.

PHP merupakan bahasa pemrograman sisi server (*server-side scripting*) yang dirancang khusus untuk membangun aplikasi web dinamis. Bahasa pemrograman ini dapat diintegrasikan dengan HTML serta didukung oleh berbagai sistem manajemen basis data seperti MySQL, PostgreSQL, SQLite, dan MariaDB.

Pada penelitian ini, PHP digunakan sebagai bahasa pemrograman utama melalui Framework Laravel dalam membangun Sistem Absensi Guru Berbasis QR Code.

---

# Pengertian PHP

PHP (Hypertext Preprocessor) merupakan bahasa pemrograman server-side yang digunakan untuk mengembangkan aplikasi web dinamis. Script PHP dijalankan pada web server, kemudian hasilnya dikirimkan ke browser dalam bentuk halaman web.

Menurut Sidik (2018), PHP merupakan bahasa pemrograman yang dirancang untuk mempermudah pembangunan aplikasi web karena memiliki sintaks yang sederhana, mudah dipelajari, dan mendukung berbagai sistem operasi maupun web server.

Sementara itu, dokumentasi resmi PHP menjelaskan bahwa PHP adalah bahasa pemrograman open source yang dapat digunakan untuk mengembangkan aplikasi web modern dengan performa tinggi serta mudah diintegrasikan dengan berbagai teknologi.

---

# Sejarah PHP

PHP pertama kali dikembangkan oleh **Rasmus Lerdorf** pada tahun 1994. Awalnya PHP hanya digunakan untuk mengelola halaman web pribadi (*Personal Home Page Tools*).

Seiring berkembangnya teknologi web, PHP mengalami banyak pembaruan hingga menjadi salah satu bahasa pemrograman web yang paling populer di dunia. Saat ini PHP digunakan oleh berbagai framework modern seperti Laravel, CodeIgniter, Symfony, dan CakePHP.

---

# Karakteristik PHP

PHP memiliki beberapa karakteristik, yaitu:

- Bersifat open source.
- Dijalankan pada sisi server (*server-side scripting*).
- Dapat diintegrasikan dengan HTML.
- Mendukung berbagai sistem operasi.
- Mendukung berbagai jenis basis data.
- Memiliki komunitas pengembang yang besar.
- Mudah dipelajari dan digunakan.

---

# Fungsi PHP

PHP memiliki berbagai fungsi dalam pengembangan aplikasi web, antara lain:

1. Mengolah data dari pengguna.
2. Menghubungkan aplikasi dengan basis data.
3. Mengelola proses autentikasi pengguna.
4. Membuat halaman web dinamis.
5. Memproses formulir (*form*).
6. Mengelola sesi (*session*) dan *cookie*.
7. Menghasilkan laporan berdasarkan data yang tersimpan.

---

# Kelebihan PHP

PHP memiliki beberapa kelebihan, yaitu:

- Gratis dan bersifat open source.
- Mudah dipelajari.
- Dokumentasi sangat lengkap.
- Kompatibel dengan berbagai web server.
- Mendukung banyak database.
- Memiliki performa yang baik.
- Didukung oleh berbagai framework modern.

---

# Kekurangan PHP

Selain memiliki kelebihan, PHP juga memiliki beberapa kekurangan, yaitu:

- Membutuhkan konfigurasi server untuk menjalankan aplikasi.
- Apabila penulisan kode tidak terstruktur, aplikasi menjadi sulit dipelihara.
- Memerlukan perhatian terhadap aspek keamanan dalam proses pengembangan.

---

# Penerapan PHP dalam Penelitian

Pada penelitian ini, PHP digunakan sebagai bahasa pemrograman utama melalui Framework Laravel. Seluruh logika aplikasi, seperti proses login, pengelolaan data guru, pembuatan QR Code, proses absensi, hingga penyusunan laporan, dikembangkan menggunakan PHP.

Penggunaan PHP bersama Laravel mempermudah proses pengembangan karena menyediakan struktur kode yang lebih terorganisasi, mendukung konsep MVC, serta mempermudah integrasi dengan database MySQL.

---

# Kesimpulan

PHP merupakan bahasa pemrograman server-side yang banyak digunakan dalam pengembangan aplikasi web. Dengan dukungan komunitas yang besar, dokumentasi lengkap, serta kompatibilitas dengan berbagai framework modern seperti Laravel, PHP menjadi pilihan yang tepat dalam pembangunan Sistem Absensi Guru Berbasis QR Code pada penelitian ini.
