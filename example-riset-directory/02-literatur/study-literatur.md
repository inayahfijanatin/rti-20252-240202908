# Studi Literatur

Dokumen ini berisi hasil kajian literatur yang menjadi dasar dalam penelitian **Perancangan dan Implementasi Sistem Absensi Guru Berbasis QR Code Menggunakan Framework Laravel**.

Studi literatur dilakukan dengan mengumpulkan, mempelajari, dan menganalisis berbagai buku, jurnal ilmiah, prosiding, serta referensi lain yang berkaitan dengan sistem informasi, teknologi QR Code, Framework Laravel, metode pengembangan perangkat lunak, dan penelitian terdahulu yang relevan. Hasil kajian ini digunakan sebagai dasar dalam penyusunan proposal, perancangan sistem, implementasi aplikasi, serta analisis hasil penelitian.

---

# Ringkasan Studi Literatur

Perkembangan teknologi informasi telah mendorong berbagai institusi pendidikan untuk menerapkan sistem informasi berbasis web dalam mendukung aktivitas administrasi sekolah, termasuk proses pencatatan kehadiran guru.

Berdasarkan hasil studi literatur, penerapan teknologi QR Code mampu meningkatkan kecepatan proses absensi dibandingkan metode pencatatan manual. Selain itu, penggunaan Framework Laravel memberikan kemudahan dalam proses pengembangan aplikasi karena menerapkan konsep Model-View-Controller (MVC), memiliki sistem routing yang baik, serta menyediakan berbagai fitur keamanan dan pengelolaan basis data.

Berbagai penelitian terdahulu menunjukkan bahwa sistem absensi berbasis QR Code mampu meningkatkan efisiensi administrasi, mengurangi kesalahan pencatatan, serta mempermudah proses penyusunan laporan kehadiran guru.

---

# Research Gap

Beberapa penelitian terdahulu telah berhasil mengembangkan sistem absensi berbasis QR Code dengan berbagai pendekatan dan teknologi.

Namun demikian, masih ditemukan beberapa keterbatasan, antara lain:

- Sistem belum menggunakan Framework Laravel versi terbaru.
- Tampilan antarmuka belum optimal untuk kebutuhan administrasi sekolah.
- Belum tersedia fitur pengelolaan data guru yang terintegrasi.
- Proses pembuatan laporan absensi masih terbatas.
- Belum seluruh sistem menerapkan pengujian Black Box secara menyeluruh.

Berdasarkan kondisi tersebut, penelitian ini mengembangkan sistem absensi guru berbasis QR Code menggunakan Framework Laravel dengan pendekatan metode Waterfall sehingga menghasilkan sistem yang lebih terstruktur, mudah dikembangkan, dan sesuai dengan kebutuhan pengguna.
