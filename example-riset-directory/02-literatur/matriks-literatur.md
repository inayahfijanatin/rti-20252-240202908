# Matriks Literatur

Dokumen ini berisi ringkasan dan perbandingan penelitian terdahulu yang menjadi referensi dalam penelitian **Perancangan dan Implementasi Sistem Absensi Guru Berbasis QR Code Menggunakan Framework Laravel**.

Matriks literatur digunakan untuk mengidentifikasi persamaan, perbedaan, kelebihan, kekurangan, serta *research gap* yang menjadi dasar pengembangan penelitian ini.

---

# Tujuan Penyusunan Matriks Literatur

Penyusunan matriks literatur bertujuan untuk:

- Mengidentifikasi penelitian yang relevan dengan topik penelitian.
- Membandingkan metode, teknologi, dan hasil penelitian terdahulu.
- Menentukan posisi penelitian yang dilakukan.
- Menemukan *research gap* sebagai dasar pengembangan sistem.

# Analisis Penelitian Terdahulu

Berdasarkan penelitian terdahulu yang telah dikaji, dapat disimpulkan bahwa:

- Teknologi QR Code telah banyak diterapkan pada sistem absensi.
- Framework Laravel banyak digunakan dalam pengembangan sistem informasi berbasis web.
- Sebagian besar penelitian menggunakan metode Waterfall dalam proses pengembangan perangkat lunak.
- Penggunaan QR Code mampu meningkatkan efisiensi proses pencatatan kehadiran dibandingkan metode manual.

Namun demikian, masih terdapat beberapa aspek yang dapat dikembangkan lebih lanjut, seperti integrasi fitur pengelolaan data guru, penyajian laporan absensi yang lebih lengkap, serta implementasi antarmuka yang lebih responsif.

# Research Gap

Perbedaan penelitian ini dengan penelitian terdahulu antara lain:

| Penelitian Terdahulu | Penelitian Ini |
|-----------------------|----------------|
| Fokus pada sistem absensi berbasis web secara umum. | Fokus pada sistem absensi guru berbasis QR Code. |
| Sebagian penelitian menggunakan framework selain Laravel. | Menggunakan Framework Laravel sebagai platform utama. |
| Fitur laporan masih sederhana. | Menyediakan pengelolaan data guru, absensi, QR Code, dan laporan dalam satu sistem. |
| Dokumentasi implementasi terbatas. | Menyediakan dokumentasi penelitian dan implementasi secara lengkap melalui repository GitHub. |

---

# Kesimpulan

Hasil kajian penelitian terdahulu menunjukkan bahwa penggunaan QR Code pada sistem absensi mampu meningkatkan efektivitas proses pencatatan kehadiran. Berdasarkan hasil analisis tersebut, penelitian ini mengembangkan sistem absensi guru berbasis QR Code menggunakan Framework Laravel dengan tujuan menghasilkan sistem yang lebih terintegrasi, mudah digunakan, dan sesuai dengan kebutuhan administrasi sekolah.
